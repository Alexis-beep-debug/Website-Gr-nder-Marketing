# Bilder & Assets

## Ordnerstruktur

### `/team`
Fotos von Malte-Valentin Gründer hier ablegen.

**Empfohlene Dateien:**
- `malte.jpg` oder `malte.webp` – Hauptfoto (wird auf der "Über uns"-Sektion angezeigt)
- Format: **JPG oder WebP**
- Mindestgröße: **800x1000px** (Hochformat, 4:5)
- Dateigröße: möglichst unter **500 KB** (für schnelle Ladezeiten)

### `/logos`
Firmenlogos hier ablegen.

**Empfohlene Dateien:**
- `logo.svg` – Hauptlogo (SVG bevorzugt für Schärfe auf allen Bildschirmen)
- `logo-white.svg` – Logo in Weiß (für dunkle Hintergründe, z.B. Footer)
- `favicon.ico` oder `favicon.svg` – Browser-Tab-Icon (mind. 32x32px)

## Nach dem Hochladen

Die Bilder werden automatisch in die Website eingebunden, sobald sie mit den richtigen Dateinamen hier abgelegt werden.
